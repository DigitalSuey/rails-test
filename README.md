# RAILS test/tutorials

<img src="https://img.shields.io/badge/Ruby-2.4.4-red.svg?style=flat" alt="Ruby" />

### Setup app
```bash
$ gem install
$ bundle install
$ yarn install
```

### Run app locally:
```bash
$ rails server

# new terminal tab:
$ ./bin/webpack-dev-server
```
View website on localhost://3000
